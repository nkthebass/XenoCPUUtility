# XenoCPUUtility
![Status](https://img.shields.io/badge/status-active-success)

> **Official Repository:** [https://github.com/nkthebass/XenoCPUUtility](https://github.com/nkthebass/XenoCPUUtility)  
> **Author:** nkthebass

XenoCPUUtility is a high-precision **CPU benchmarking and stress testing utility** designed to measure single-core and multi-core performance, thread scaling, floating-point throughput, and sustained CPU stability.

### üöÄ Key Features
* **Deterministic Benchmarking:** Consistent workloads for reliable scoring.
* **Scaling Tests:** Full support for single-core and multi-core scaling.
* **High Thread Count:** Configurable up to 64 threads (32 cores).
* **Workload Coverage:** Extensive floating-point and integer testing.
* **Thermal Optimization:** 8-second heavy load "warmup" and enhanced between-run delays for thermal recovery.
* **Outlier Detection:** Visual highlighting for anomalous runs to ensures data integrity.
* **Memory Testing:** RAM stress testing from 256MB to 128GB.
* **Modern UI:** Lightweight footprint (no background services) with smooth gradients and animations.
* **Architecture Support:** Designed specifically for Intel, AMD, and ARM.
